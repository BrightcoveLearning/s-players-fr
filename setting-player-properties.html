Pour modifier les propriétés du lecteur, procédez comme suit:

  1. Ouvrez le module Players.
  2. Cliquez sur le nom d'un lecteur .

La page des propriétés du lecteur s'ouvre. Les boutons en haut de la page permettent de:

  • Bouton Plus
    • Supprimer un lecteur
    • Activer/désactiver un lecteur
    • Annulation des modifications
  • Prévisualiser un lecteur
  • Publiez et copiez le code intégré d'un lecteur

Pour modifier les propriétés du lecteur, cliquez sur un élément dans le menu de navigation de gauche. Les propriétés du lecteur suivantes peuvent être configurées:

  • Informations sur le lecteur
  • coiffant
  • Lecture
  • Contrôles
  • Social
  • Plugins
  • Écrans de fin
  • Publicité
  • Restriction relative
  • Campagne
  • Éditeur JSON
  • Analyses tierces
Remarque: Toute modification apportée à un lecteur peut prendre jusqu'à 5 minutes avant d'apparaître dans les lecteurs publiés.

Suppression d'un lecteur

La suppression d'un lecteur le supprime de votre compte et ne peut pas être annulé. Pour supprimer un lecteur :

  1. Cliquez sur Plus... > Supprimer.
  2. Confirmez la suppression en cliquant sur Supprimer.

Activation/désactivation d'un lecteur

Les lecteurs activés peuvent lire des vidéos. Les lecteurs désactivés ne peuvent pas lire les vidéos. Pour activer/désactiver un lecteur :

  1. Cliquez sur Plus... > Activer ou désactiver.
  2. Confirmez l'activation ou la désactivation.

Annulation des modifications

Le module Players permet d'annuler les changements de lecteur s. L'annulation des modifications annulera toutes les modifications non publiées apportées à un lecteur . Pour annuler les modifications apportées à un lecteur:

  1. Cliquez sur Plus... > Annulation des modifications.
  2. Confirmez l'action en cliquant sur Ignorer les modifications.

Prévisualisation d'un lecteur

Le module Players permet de prévisualiser les modifications apportées au lecteur avant leur publication. Cela vous permet de voir les modifications en action avant de publier le lecteur sur un site de production. Pour prévisualiser un lecteur, cliquez sur Aperçu. La boîte de dialogue d'aperçu fournit:

  • URL de prévisualisation
  • Code d'intégration du lecteur Standard et Avancé
Notez que l'URL de prévisualisation et le code intégré ne doivent pas être utilisés sur les sites de production.

Publier et récupérer le code intégré d'un lecteur

Le bouton Publish & Embed permet de publier le lecteur et de générer le code d'intégration du lecteur et l'URL du lecteur publié. Pour publier un lecteur et obtenir le code intégré:

  1. Cliquez sur Publier et intégrer.
  2. Cliquez sur Publier les modifications.

Une fois qu'un lecteur est publié, les modifications seront répercutées dans toutes les instances de ce lecteur en particulier. L'affichage des modifications dans les lecteurs publiés peut prendre quelques minutes.

Le code d'intégration standard (iframe) et avancé (intégré à la page) est fourni. Pour plus d'informations sur les différences entre le lecteur d'aperçu et le lecteur publié et sur la génération du code intégré pour un lecteur, consultez Génération du code intégré du lecteur.

Informations sur le lecteur

Les propriétés Informations sur le lecteur permettent de configurer le nom du lecteur, sa taille, son format, son format, son mode de mise à jour et sa version. Pour plus d'informations, consultez la section Définition des propriétés des informations du lecteur.

coiffant

Les propriétés Styling sont utilisées pour configurer le type de lecteur, l'apparence du lecteur, les couleurs et une superposition de lecteur. Pour plus d'informations, consultez Styling Players.

Lecture

Les propriétés Playback sont utilisées pour configurer les options de lecteur vidéo. Pour plus d'informations, consultez la section Configuration des propriétés de lecteur du lecteur.

Contrôles

Les propriétés Controls sont utilisées pour configurer les commandes qui apparaissent dans la barre de contrôle du lecteur. Pour plus d'informations, consultez Configuration des commandes du lecteur.

Social

Les propriétés sociales sont utilisées pour configurer les options sociales du lecteur. Pour plus d'informations, consultez Configuration des propriétés sociales d'un lecteur.

Plugins

Les plugins utilisent une combinaison de HTML, JavaScript et/ou CSS pour personnaliser le comportement du lecteur. Les plugins peuvent être développés pour:

  • Modifier le comportement par défaut d'un lecteur
  • Ajout de fonctionnalités à un lecteur
  • Personnalisez l'apparence du lecteur

Pour plus d'informations, consultez Configuration des plug-ins de lecteur.

Écrans de fin

Les propriétés des écrans de fin sont utilisées pour configurer ce qui s'affiche lorsqu'une vidéo se termine. Pour plus d'informations, consultez la section Configuration de l'écran de fin du lecteur.

Publicité

Les propriétés Advertising sont utilisées pour activer les publicités IMA dans un lecteur. Pour plus d'informations, consultez Configuration de la publicité des lecteur s.

Restriction relative

Les propriétés Content Restriction sont utilisées pour configurer les restrictions de domaine et le géo-filtrage pour le lecteur. Pour plus d'informations, consultez la section Configuration des restrictions de contenu du lecteur.

Campagne

Les propriétés de la campagne sont utilisées pour intégrer les lecteurs Brightcove aux plateformes d'automatisation du marketing et aux formulaires de prospects. Éditeur JSON

L'éditeur JSON peut être utilisé pour modifier la configuration du lecteur. Il faut être prudent lors de la modification de la configuration du lecteur.

Remarque: Il s'agit d'une option avancée qui ne doit être utilisée que par les éditeurs qui sont à l'aise pour modifier la configuration du lecteur.

Pour plus d'informations sur la modification de la configuration du lecteur, consultez le Guide de configuration du lecteur.

Analyses tierces

Les propriétés Third-Third-Party Analytics sont utilisées pour intégrer Brightcove Players à Google Analytics et Adobe Analytics. Pour plus d'informations, consultez Configuration du suivi des événements Google Analytics et Configuration du suivi des événements d'Adobe Analytics.
