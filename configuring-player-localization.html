Les lecteurs Brightcove peuvent être localisés afin que les éléments de l'interface apparaissent dans différentes langues. Si les propriétés de localisation du lecteur ne sont pas configurées, l'interface s'affichera en anglais.

Pour configurer la localisation du lecteur, procédez comme suit.

  1. Ouvrez le module Players.
  2. Cliquez sur un lecteur pour ouvrir ses propriétés.
  3. Cliquez sur Informations sur le lecteur dans le menu de navigation de gauche.
  4. Dans le champ Localisation, saisissez le code de langue à deux lettres correspondant à la langue et appuyez sur Entrée. Au fur et à mesure que vous tapez, la liste des langues sera filtrée. Vous pouvez également cliquer sur une valeur pour la sélectionner.
    Remarque: La liste des langues prises en charge se trouve sur la page de langue Video.js.
  5. Ajoutez des langues supplémentaires selon vos besoins. Si un code de langue non valide est saisi, il apparaît en rouge.
  6. Cliquez sur Enregistrer

Veillez à publier toutes les modifications apportées au lecteur.

Vérification de la langue

Pour utiliser la langue localisée, définissez la langue préférée dans le navigateur. Les étapes exactes varient en fonction du navigateur et du système d'exploitation. Si la langue préférée du navigateur est l'anglais, ou une langue non disponible, le texte par défaut, ou anglais, s'affichera. Les navigateurs n'ont pas de détection des paramètres régionaux parfaitement standardisée, ce qui peut être un peu compliqué. D'une manière générale, les navigateurs cherchent à déterminer la langue souhaitée dans cet ordre:

  • Vérifiez si la page d'intégration possède un attribut lang sur l'élément <html>
  • Vérifiez les configurations locales spécifiques au navigateur
  • Revenez à l'anglais
Remarque description: Cette technique ne fonctionnera qu'avec l'implémentation de code avancé (in-page). Étant donné que l'implémentation Standard se charge dans un iframe, le paramètre de langue du navigateur n'y est pas intégré.

Si vous utilisez l'attribut HTML, vous n'avez pas besoin de définir la langue préférée par défaut dans votre navigateur ou votre ordinateur, car le lecteur utilisera celle de l'attribut lang.

<html lang="es">

De plus, si vous utilisez une iframe, n'oubliez pas que vous devrez ajouter le paramètre de requête à l'URL du lecteur.

//players.brightcove.net/6045682556001/JR6QQMDSB_Default/index.html ? videoId=6054419564001&language=es
