Les propriétés de restriction de contenu du lecteur peuvent être utilisées pour restreindre un lecteur à certains pays ou bloquer un lecteur de certains pays. Des restrictions de domaine peuvent être ajoutées à un lecteur pour limiter les domaines dans lesquels le lecteur peut être utilisé.

Remarques:

  • Les propriétés Geo Restriction ne sont disponibles que pour les comptes Enterprise et Professional. La restriction de domaine est disponible pour tous les comptes.
  • L'adresse IP peut également être utilisée pour limiter la lecteur. Pour plus d'informations, voir Restreindre la lecteur vidéo à l'aide d'une adresse IP.

Pour afficher et modifier les propriétés de restriction de contenu d'un lecteur, ouvrez le module Players, cliquez sur le nom d'un lecteur , puis cliquez sur Restriction de contenu dans le menu de navigation de gauche.

Configuration des restrictions de domaine

Des restrictions de domaine peuvent être définies sur les lecteur s afin de limiter les domaines dans lesquels les lecteur s peuvent être utilisés. Si le code de publication du lecteur est copié et utilisé sur un autre site, les restrictions de domaine empêcheront le lecteur de charger des vidéos. Les éditeurs doivent utiliser le code intégré à la page pour que la lecteur fonctionne réellement sur les domaines qui ont été mis sur liste blanche. Les éditeurs ne peuvent pas utiliser le code intégré iframe sur un domaine de la liste blanche (ou tout autre domaine) et s'attendre à ce que la lecteur fonctionne correctement.

Remarques:

  • Alors que les restrictions de domaine peuvent aider à empêcher la distribution non autorisée de votre contenu, les clients qui recherchent une sécurité maximale du contenu devraient plutôt se tourner vers des solutions DRM. Consultez Gestion des droits numériques (DRM) dans Video Cloud pour plus d'informations.
  • Les restrictions de domaine sont prises en charge lors de l'utilisation d'une expérience dans la page Gallery et du code intégré recommandé (JavaScript).
  • Les lecteurs à domaine restreint ne sont pas compatibles avec les intégrations AMP. En effet, l'intégration AMP crée une iframe qui, à son tour, n'est pas prise en charge par les lecteurs restreints au domaine.
  • Lorsque vous utilisez la fonctionnalité de restriction de domaine, les restrictions de domaine doivent être configurées pour tous les lecteur s de votre compte. Si tous les lecteurs ne sont pas configurés avec des restrictions de domaine, quelqu'un peut prendre la clé de stratégie d'un lecteur non protégé et l'utiliser pour lire n'importe quelle vidéo.
  • Bien qu'il n'y ait aucune limite quant au nombre de domaines pouvant être ajoutés dans Studio, il existe une limite sur la longueur de l'URL (qui est définie par plusieurs facteurs limitatifs tels que l'appareil/le navigateur utilisé) pour la requête d'API à l'API Playback. Cette URL inclura la liste de domaines délimitée par des virgules. Veuillez noter que cette limitation est également valable si vous utilisez uniquement Studio. Par conséquent, le nombre de domaines pouvant être spécifiés est limité, mais le nombre dépendra de la longueur des noms de domaine.

Pour configurer une restriction de domaine pour un lecteur, procédez comme suit:

  1. Saisissez une ou plusieurs URL de domaine complètes (telles que http://www.brightcove.com ou https://www.brightcove.com) pour restreindre la disponibilité de votre lecteur à ces domaines particuliers. Veillez à inclure http://ou https://. Cliquez sur +.

    Remarques:

    • Les URL complètes doivent être fournies. Les caractères génériques dans les URL ne sont pas pris en charge. Les URL ne doivent pas se terminer par un /.
    • Seuls les domaines, et non les chemins spécifiques, sont pris en charge. Par exemple, vous pouvez ajouter https://www.brightcove.com, mais pas https://www.brightcove.com /documentation
  2. Cliquez sur Enregistrer pour enregistrer les modifications.

Configuration du filtrage géographique

Le filtrage géographique peut être utilisé pour contrôler quels pays peuvent ou ne peuvent pas utiliser un lecteur.

Remarques:

  • Le filtrage géographique est une fonctionnalité disponible pour les comptes Enterprise et Professional moyennant des frais supplémentaires. Contactez Brightcove si vous souhaitez mettre à niveau votre compte afin d'inclure le filtrage géographique.
  • Le filtrage géographique peut également être appliqué à des vidéos individuelles. {{site.product_short}} appliquera les politiques de filtrage géographique des lecteurs et des vidéos. Si un pays est exclu par l'un ou l'autre, la vidéo n'y sera pas disponible.

Pour définir les propriétés de filtrage géographique d'un lecteur:

  1. Sélectionnez Autoriser le visionnage depuis les pays suivants ou Interdire l'affichage depuis les pays suivants.
  2. Saisissez les noms de pays appropriés. Au fur et à mesure que vous tapez, la liste des pays sera filtrée.
  3. Cliquez sur un pays, puis appuyez sur Entrée pour l'ajouter à la liste.
  4. Cliquez sur Enregistrer.
