Étapes

  1. Dans la navigation principale, ouvrez le module Players.
  2. Cliquez sur le nom du lecteur pour créer un style.
  3. Cliquez sur Style dans le volet de navigation de gauche.
    4. Remarque: Cliquez sur l'option de style pour mettre à jour l'habillage du lecteur et déclencher un message de changement de lecteur . Cela s'applique aux nouveaux lecteur s ou aux lecteur s qui n'ont pas de valeur de skin.
  4. Choisissez d'afficher ou de masquer le titre et la description.
  5. Sélectionnez une forme de bouton de lecteur et une position de bouton de lecteur.
  6. Cliquez sur la couleur d'un thème pour ouvrir le sélecteur de couleurs.
  7. Cliquez sur une couleur du côté conduite de la commande.
  8. Cliquez sur le carré pour sélectionner une teinte de la couleur souhaitée.
  9. Le curseur situé sous le carré de couleur peut être utilisé pour définir l'opacité des commandes.
  10. Cliquez en dehors du sélecteur de couleurs pour modifier la couleur d'un autre thème.
  11. Les modifications apportées à un lecteur DOIVENT être publiées avant d'apparaître dans les lecteurs publiés. Pour publier les modifications, cliquez sur Publier et intégrer, puis sur Publier les modifications.
Remarque: Pour plus de détails sur le style des lecteur s, voir Styling Players.
