Les lecteurs peuvent être configurés pour que le contenu vidéo puisse être partagé sur des sites de réseaux sociaux populaires. Lorsque le partage social est activé, une icône de partage apparaît dans le dock titre et description.

Remarque: Si le lecteur est configuré pour ne pas afficher le titre et la description, l'icône de partage apparaît en bas à droite du lecteur, à côté des commandes de l'autre lecteur .

Lorsque vous cliquez sur l'icône de partage, les options de partage sélectionnées apparaissent pour permettre le partage de la vidéo.

Pour configurer les propriétés sociales, procédez comme suit:

  1. Ouvrez le module Players.
  2. Cliquez sur le lien correspondant au lecteur que vous souhaitez configurer.
  3. Cliquez sur Social dans le menu de navigation de gauche.
  4. Cochez la case pour activer le partage et sélectionnez les sites de réseaux sociaux sur lesquels la vidéo peut être partagée. {% if site.product_short == Video Cloud-fr/ %} {% endif %} {% if site.product_short == Beacon-fr/ %} {% endif %}
  5. Cochez Afficher le code intégré du lecteur pour permettre au lecteur de copier le code intégré du lecteur lorsque les options de partage social sont affichées.
  6. Cochez Afficher le lien direct du lecteur pour permettre au spectateur de copier un lien vers la page où le lecteur est intégré lorsque les options de partage social sont affichées.
    Remarque: Lorsque cette option est sélectionnée, le code d'intégration avancé est requis pour que l'URL complète de la page soit affichée. L'utilisation du code intégré standard n'affichera pas l'URL complète de la page.
