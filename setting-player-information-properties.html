Les propriétés des informations du lecteur peuvent être utilisées pour définir le nom du lecteur , la taille, le format, les dimensions, le mode de mise à jour et la version du lecteur.

De nombreuses propriétés du lecteur peuvent également être définies/modifiées dynamiquement en modifiant le code de publication du lecteur. Pour plus d'informations, consultez ces documents connexes selon que vous utilisez le code intégré Standard (iframe) ou Avancé (intégré à la page).

Pour configurer les propriétés des informations sur le lecteur, procédez comme suit:

  1. Ouvrez le module Players.
  2. Cliquez sur le lien correspondant au lecteur que vous souhaitez configurer.
  3. Cliquez sur Informations sur le lecteur dans le menu de navigation de gauche.

Remarque: Toute modification apportée à un lecteur peut prendre jusqu'à 5 minutes avant d'apparaître dans les lecteurs publiés.

Les propriétés d'informations de lecteur suivantes peuvent être configurées:

  • Nom et brève description
  • Type de lecteur
  • dimensionnement
  • Rapport hauteur/largeur
  • Dimensions du lecteur
  • Mode mise à jour
  • Version du lecteur
  • Localisation

Après avoir apporté des modifications, cliquez sur Enregistrer, puis publiez le lecteur.

Type de lecteur

Le paramètre Type de lecteur détermine si le lecteur affichera une seule vidéo ou si les listes de lecteur sont prises en charge:

  • Vidéo unique: une seule vidéo s'affiche
  • Playlist - Le lecteur prend en charge les listes

La sélection de Playlist activera certaines propriétés supplémentaires de style et de lecteur.

dimensionnement

La taille peut être définie sur Réactive ou Fixe.

  • Responsive - La taille du lecteur s'ajustera automatiquement en fonction de la taille du navigateur
  • Corrigé - La taille du lecteur était basée sur les paramètres de largeur et de hauteur
Remarque: Si vous sélectionnez l'option de dimensionnement réactif, le code Standard (iframe) fonctionne bien et le lecteur et la liste de lecteur sont réactifs. Si vous utilisez le code Avancé (intégré à la page), le lecteur répondra, mais pas la liste de lecteur.

Rapport hauteur/largeur

Le format d'image définit le rapport hauteur/largeur du lecteur.

  • 1:1, 3:2, 4:3, 16:9, 21:9 ou 9:16 - Entrez la largeur d'un lecteur et la hauteur sera déterminée automatiquement
  • Personnalisé: saisissez des valeurs pour la largeur et la hauteur du lecteur

Dimensions du lecteur

Les dimensions du lecteur contrôlent la largeur et la hauteur du lecteur.

Mode mise à jour et version du lecteur

Le mode de mise à jour contrôle la façon dont les mises à jour des lecteur s La version du lecteur est utilisée pour sélectionner une version du lecteur. Le champ Version du lecteur publiée affiche la version actuelle du lecteur. Pour plus d'informations sur ces paramètres et la façon dont ils doivent être utilisés, consultez Configuration du mode et de la version de mise à jour du lecteur.

Localisation

Le champ Localisation peut être utilisé pour inclure la ou les langues que vous souhaitez utiliser pour la localisation du lecteur. Pour plus d'informations sur la localisation des lecteurs, consultez Configuration de la localisation du lecteur.
