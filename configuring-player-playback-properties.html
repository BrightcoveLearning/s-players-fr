Les propriétés Playback peuvent être utilisées pour configurer la façon dont la lecteur vidéo est gérée dans le lecteur.

Pour configurer les propriétés de lecteur, procédez comme suit:

  1. Ouvrez le module Players.
  2. Cliquez sur le lien correspondant au lecteur que vous souhaitez configurer.
  3. Cliquez sur Lecture dans le menu de navigation de gauche.

Remarque: Toute modification apportée à un lecteur peut prendre jusqu'à 5 minutes avant d'apparaître dans les lecteurs publiés.

Les propriétés de lecteur suivantes peuvent être configurées:

  • Lecture automatique de la vidéo lors du chargement du lecteur
  • Jouer en mode muet
  • Précharger la vidéo avant la lecteur
  • Activer les DRM
  • Propriétés de la playlist

Lecture automatique de la vidéo lors du chargement du lecteur

Le paramètre Auto-Play Video on Player Load contrôle si la vidéo commencera à être lue lorsque le lecteur sera chargé. Ce paramètre peut fonctionner différemment selon le navigateur. Pour plus d'informations, consultez la section Considérations relatives à la lecteur automatique.

Jouer en mode muet

Le paramètre Play Muted contrôle si le son de la vidéo sera désactivé.

Précharger la vidéo avant la lecteur

Le paramètre Précharger la vidéo avant la lecteur contrôle si le lecteur doit précharger le contenu vidéo avant que l'utilisateur ne lance la lecteur.

Remarque: Pour plus de détails, consultez la section Détails du préchargement du document Options disponibles pour le document Code incorporé avancé (intégré à la page).

Les options suivantes sont disponibles:

  • Preload Video (fastest) - Indique que l'intégralité du fichier vidéo peut être téléchargée, même si l'utilisateur ne regarde pas la vidéo. Du côté positif, si le spectateur lit la vidéo, elle peut commencer à jouer immédiatement. D'un autre côté, cette option augmentera la consommation de bande passante au fur et à mesure que le précharge/chargement se produit même si le spectateur ne lit pas la vidéo.
  • Précharger les métadonnées: seules les métadonnées vidéo sont préchargées. C'est la valeur par défaut.
  • Ne pas précharger: aucune vidéo n'est préchargée

Remarques:

  • Lors de l'utilisation de l'insertion publicitaire côté serveur (SSAI), Brightcove recommande de définir preload=none. Si le préchargement est défini sur auto ou les métadonnées, les premières images du flux vidéo seront demandées, ce qui comptabilise l'annonce comme diffusée même si l'utilisateur ne démarre pas la lecteur.
  • La valeur du paramètre de préchargement dans Studio remplacera la valeur de l'attribut de préchargement dans le code intégré avancé (In-Page).

Activer les DRM

Le paramètre Activer DRM, lorsqu'il est coché, permet de lire du contenu protégé par DRM dans le lecteur.

Propriétés de la playlist

Lorsqu'un type de liste de lecteur de lecteur est sélectionné dans le cadre des propriétés Informations sur le lecteur, une lecteur supplémentaire les propriétés seront affichées.

  • Ordre de lecteur aléatoire - Les vidéos seront lues dans un ordre aléatoire
  • Lire la vidéo lorsqu'elle est sélectionnée: si vous cliquez sur une miniature dans le menu de la liste de lecteur, la vidéo sera lue lors du chargement, même si la vidéo précédente du lecteur a été mise en pause
  • Mode de lecteur continue - Les vidéos avancent automatiquement et se répètent indéfiniment
  • Compte à rebours vidéo - Activé uniquement lorsque le mode de lecteur continue est coché
    • Carte suivante (avance automatique) - Une carte suivante s'affiche dans le lecteur
    • Écran de fin suivant (compte à rebours) - Un compte à rebours de l'écran de fin s'affiche dans le lecteur
